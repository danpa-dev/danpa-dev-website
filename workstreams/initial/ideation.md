# Ideation Notes

Goal: personal homepage hosted on GitHub Pages with custom domain, links to projects/LinkedIn, and a blog section.

Open questions and decisions:
- Site generator: plain HTML/CSS, static site generator (e.g., Eleventy, Jekyll), or framework (e.g., Astro)?
- Blog workflow: markdown posts in repo vs. external CMS.
- Content: bio, featured projects, contact method, and visual style.
- Deployment: user site vs. project site on GitHub Pages; custom domain (Squarespace-managed DNS).
